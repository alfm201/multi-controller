# CLAUDE.md

## 프로젝트 개요

Multi Screen Pass — 여러 PC 간 마우스/키보드를 경계 기반으로 전환하는 Windows 유틸리티.
Python + PyQt5, Win32 low-level hook 기반.

## 코드 수정 원칙

- 기존 코드 스타일을 따른다.
- 요청 범위 밖의 리팩터링은 하지 않는다.
- 버그 수정 시 재현 테스트를 먼저 시도하고, 수정 후 같은 테스트로 검증한다.
- 에러는 로그와 스택트레이스를 근거로 수정한다.

## 실행 규칙

- 변경 후에는 가장 좁은 범위의 테스트부터 실행하고, 통과하면 필요 시 범위를 넓힌다.
- 한글 파일/UI 문구 확인은 PowerShell 출력을 믿지 않는다.
  Python으로 `encoding='utf-8'` 직접 읽는 방식을 우선한다.

## 응답 방식
